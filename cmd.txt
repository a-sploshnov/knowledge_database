dir - список всех файлов в папке
dir + имя папки - просмотр содержимого папки
mkdir + имя - создать папку в текущем каталоге
rmdir + имя - удалить папку
cls - очистить рабочую область командной строки
del + имя - удалить файл
start + имя - открыть папку
имя файла с расширением - открыть файл
cd + имя - переход в каталог
cd .. - перейти на каталог вверх
cd /D + путь - переход на другой диск
